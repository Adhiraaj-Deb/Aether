# Aether - Interactive Music Visualizer

A stunning, glassmorphism-styled 3D music visualizer built with Three.js. Features audio-reactive liquid crystal geometry, DNA helix particles, and a black hole simulations.

## ‚ú® Features

-   **Visual Effects:**
    -   **Liquid Crystal**: Iridescent, organic morphing core.
    -   **Black Hole**: Gravitational lensing and accretion disk simulation.
    -   **DNA Helix**: Dual-strand spiral particles representing frequency bands.
-   **Interactive UI:**
    -   **Glassmorphism Player**: Frosted glass music controls.
    -   **Settings Menu**: Customize everything from particle count to bloom intensity.
    -   **Touch/Mouse Reactivity**: Background particles ripple away from your touch.
-   **Audio Analysis**:
    -   Real-time frequency analysis.
    -   Beat detection and transient response.
